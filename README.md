# Sales-Data-Analyst
# ☕ Afficionado Coffee Roasters - Sales Trend & Time-Based Performance Analysis

## 📋 Project Overview
This project analyzes 149,116 transactions from Afficionado Coffee Roasters across 3 store locations (Lower Manhattan, Hell's Kitchen, Astoria) to identify peak sales hours, day-of-week patterns, and store-level performance — enabling data-driven staffing and operational decisions.

**Internship:** Unified Mentor  
**Author:** Yashika

## 🎯 Objectives
- Identify overall sales trends across 2025
- Determine busiest and slowest transaction hours
- Compare revenue performance across store locations
- Build interactive dashboards for stakeholders

## 🛠️ Tools & Technologies
- **Python** (Pandas, Matplotlib, Seaborn) — Data cleaning, EDA, feature engineering
- **Excel** — Pivot tables & charts
- **Tableau** — Interactive visualizations ([Live Dashboard](your-tableau-public-link))
- **Power BI** — Executive dashboard
- **Streamlit** — Live interactive web app ([Live App](your-streamlit-link))

## 📊 Key Insights
- **Peak hours:** 8 AM – 10 AM (morning coffee rush)
- **Morning (6-11)** generates highest revenue: **$388,288**
- **Hell's Kitchen** is the top-performing store: **$236,511**
- Revenue drops sharply after 6 PM across all locations
- Total Revenue: **$698,812** | Total Transactions: **149,116**
